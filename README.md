# IDfy GIG Report — React App

## Getting Started
```bash
npm install
npm start
```

## Sections & Animations

### 1. Hero Section — Scroll animation
- 16 BW collage cards on a horizontal oval
- Cards fly off radially as you scroll
- Dark red/black gradient bg

### 2. Stats Section
- Count-up numbers (7.7M → 23.5M) trigger on scroll-into-view
- 4 pulsing dot stat indicators
- **"still remains..."** — dots appear one-by-one in a loop animation

### 3. Pillars Section — No thread
- 3 clean cards (SPEED / SCALE / TRUST) fade+rise into view
- Red pulsing dot header on each card
- BW gear/road illustration inside each

### 4. Trust Section — Scroll-driven text reveal
- Each line ("Trust,", "however is", "fragile.", "One news headline.", etc.)
  slides up from below as you scroll through a sticky 350vh section
- Side annotation bubbles appear mid-scroll

### 5. Why Report Section
- Left: 4.9M count-up stat
- Right: grey box with 4 bullet-point aims

### 6. Workforce Section — Cards slide from right
- Sticky 280vh section split white/dark
- 6 timeline cards (1999–2025) each slide in from the right with stagger
- Smooth ease-in per card as scroll progresses
